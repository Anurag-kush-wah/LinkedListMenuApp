# LinkedListMenuApp

A **menu-driven Java program** to perform basic **LinkedList operations** using Java's built-in `LinkedList` class.

---

## Features

- Insert elements into the linked list
- Delete elements from the linked list
- Display all elements
- Menu-driven interface
- Handles empty list and invalid input scenarios

---

## Technology

- Java (JDK 8+)
- Java Collections Framework (`LinkedList`)
- Scanner class for input
